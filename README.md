# HR Analytics Dashboard

## Project Overview
This Power BI dashboard analaise employee attrition patterns and workforce demographics to help organisations understand factors influencing employee turnover.

## Tools Used
- Power BI
- Microsoft Excel
- Data Visualisation

## Key Metrics
- Total Employees
- Attrition Count
- Average Age

## Dashboard Insights
- Attrition by Department
- Attrition by Gender
- Attrition by Job Role
- Attrition by Age
- Attrition by Education Field

## Features
- Interactive Slicers
- KPI Cards
- Dynamic Filtering
- Business Insights

## Business Impact
This dashboard helps HR teams identify employee retention challenges and make data driven workforce decisions.
